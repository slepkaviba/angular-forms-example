Forms in Angular

Create a form that takes three input vaules:

  • Name
  • Age
  • Birthdate

Form should calculate real user's age in the background based on the birthdate.

Form submission is allowed only for:

  • Names longer than 2 letters
  • Age above or equal to 18

Submission shows form's payload in the dialog.

User can reset the form if and only if any changes were made.
